Use the `generate_diagram` MCP tool to create the user's diagram in FigJam.

Supported: flowcharts, sequence diagrams, state diagrams (stateDiagram-v2), Gantt charts.
NOT supported: class diagrams, timelines, venn diagrams, ER diagrams.

If the user hasn't described what diagram they want, ask first.
After creation, tell the user the diagram was created in FigJam and how to edit it there.
